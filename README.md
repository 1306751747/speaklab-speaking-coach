# SpeakLab 英语口语练习工具

SpeakLab 是一款面向英语口语场景训练的 Web 原型，帮助用户在面试、点餐、会议、旅行等指定场景下进行模拟真实对话。应用支持浏览器语音识别、英文语音教练、即时纠错、发音/流利度/语法/场景贴合评分，以及课后可量化总结。

## 功能

- 场景选择：面试、点餐、会议、旅行，每个场景包含独立对话目标和追问脚本。
- 实时语音对话：使用浏览器 Web Speech API 进行英文语音识别与语音合成，也支持键盘输入兜底。
- 发音评测：结合语音识别置信度、回答长度、填充词和语速估算发音清晰度与流利度。
- 语法/表达纠错：在用户每轮回答后即时展示可执行修改建议，避免打断用户正在说话。
- 课后总结：完成至少 3 轮对话后，输出综合均分、维度分数、填充词数量、高频问题和下一轮训练重点。

## 本地运行

无需安装依赖，直接用现代浏览器打开：

```bash
app/index.html
```

如果浏览器限制本地文件的麦克风权限，可启动一个静态服务：

```bash
python -m http.server 8080 -d app
```

然后访问 `http://localhost:8080`。推荐使用 Chrome 或 Edge，并允许麦克风权限。

## 技术实现

- 前端：HTML、CSS、原生 JavaScript。
- 语音识别：`SpeechRecognition` / `webkitSpeechRecognition`。
- 语音合成：`SpeechSynthesisUtterance`。
- 对话引擎：本地场景脚本 + 回合状态机，保证离线可演示。
- 评测模型：基于识别置信度、WPM、填充词、关键词命中、规则纠错数量进行加权评分。
- 纠错策略：每轮回答结束后触发，不在用户说话过程中打断；输出“错误表达 -> 推荐表达 -> 原因”。

## 依赖与原创说明

本原型未引入第三方库或历史代码。核心功能代码均位于 `app/app.js`，样式位于 `app/styles.css`。浏览器 Web Speech API 属于运行环境能力，不是随项目打包的第三方依赖。

若后续接入在线大模型或流式语音服务，应在 PR 描述和 README 中补充服务商、SDK、模型、费用和数据隐私说明。

## PR 提交规范建议

每个 PR 只做一件事，建议拆分如下：

1. `feat: add scenario selection shell`
   - 功能描述：添加训练场景列表和基础页面框架。
   - 实现思路：使用静态场景配置驱动 UI。
   - 测试方式：切换场景后标题和首轮问题正确变化。
2. `feat: add speech conversation loop`
   - 功能描述：支持麦克风输入、语音合成和逐轮对话。
   - 实现思路：封装 Web Speech API，维护回合状态。
   - 测试方式：允许麦克风权限后可识别英文并发送回答。
3. `feat: add speaking assessment`
   - 功能描述：展示发音、流利度、语法、场景贴合评分。
   - 实现思路：按置信度、语速、关键词和纠错数量计算分数。
   - 测试方式：输入不同长度和错误表达后，分数与纠错变化。
4. `feat: add after-class summary`
   - 功能描述：完成 3 轮后生成可量化课后总结。
   - 实现思路：聚合每轮评分、词数、填充词和纠错记录。
   - 测试方式：完成 3 轮后点击生成总结。

## 后续增强方向

- 接入低延迟流式语音模型，实现更自然的端到端实时对话。
- 使用音素级发音评测服务，替代当前浏览器识别置信度代理指标。
- 引入 CEFR 或 IELTS 口语 rubric，生成更标准化的能力成长曲线。
- 保存训练历史，展示周维度进步趋势和高频错误清单。
